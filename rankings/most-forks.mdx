The Most Forks ranking tracks the 100 most-forked repositories on GitHub, regardless of programming language. Rankings are determined by total fork count and are refreshed automatically.

## GraphQL query

Data is retrieved using the GitHub GraphQL API v4 with the following search query:

```graphql
forks:>1000 sort:forks
```

This returns repositories with more than 1,000 forks, sorted in descending order by fork count. Results are fetched in two batches of 50 for a total of 100 repositories.
