## Ranking fields

Each ranking table includes the following fields:

| Field | Description |
| ----- | ----------- |
| Ranking | Position within the top 100 list |
| Project Name | Repository name, linked to the GitHub repository |
| Stars | Total number of stargazers |
| Forks | Total number of forks |
| Language | Primary programming language detected by GitHub |
| Open Issues | Number of currently open issues |
| Description | Repository description from GitHub |
| Last Commit | Timestamp of the most recent push to the repository |
