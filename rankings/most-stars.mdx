The Most Stars ranking tracks the 100 most-starred repositories on GitHub, regardless of programming language. Rankings are determined by total stargazer count and are refreshed automatically.

## GraphQL query

Data is retrieved using the GitHub GraphQL API v4 with the following search query:

```graphql
stars:>1000 sort:stars
```

This returns repositories with more than 1,000 stars, sorted in descending order by star count. Results are fetched in two batches of 50 for a total of 100 repositories.
