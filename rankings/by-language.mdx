Language rankings track the 100 most-starred repositories for each of 34 programming languages. Each language uses a dedicated query scoped to that language and ranked by star count.

## GraphQL query

Data is retrieved using the GitHub GraphQL API v4. For each language, the search query follows this pattern:

```graphql
language:{lang} stars:>0 sort:stars
```

For example, to retrieve the top Python repositories:

```graphql
language:Python stars:>0 sort:stars
```

Results are fetched in two batches of 50 for a total of 100 repositories per language.
