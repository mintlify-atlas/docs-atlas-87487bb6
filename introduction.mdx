GitHub Ranking is an open-source data platform that automatically collects and publishes the most-starred and most-forked repositories on GitHub. Updated daily using the GitHub GraphQL API, it tracks 34 programming languages and maintains historical CSV snapshots going back to 2018.

## What's tracked

GitHub Ranking fetches the **Top 100** repositories for each category:

- **Most Stars** — the most-starred repositories across all languages
- **Most Forks** — the most-forked repositories across all languages
- **34 Language Rankings** — including Python, JavaScript, TypeScript, Go, Rust, Java, C, C++, and more

<Info>
  Rankings are updated automatically every day. The last update timestamp is displayed at the top of the [README](https://github.com/EvanLi/Github-Ranking).
</Info>

## Key features

<CardGroup cols={2}>
  <Card title="Daily auto-updates" icon="rotate">
    Cron job runs `process.py` daily and commits updated rankings to the repository
  </Card>
  <Card title="GitHub GraphQL API v4" icon="code">
    Uses GraphQL for efficient, paginated queries — up to 100 results per request
  </Card>
  <Card title="34 language categories" icon="language">
    ActionScript, C, C#, C++, Clojure, Go, Java, JavaScript, Python, Rust, TypeScript, and many more
  </Card>
  <Card title="Historical CSV data" icon="table">
    Every daily snapshot is saved as a CSV file in the `Data/` directory since December 2018
  </Card>
</CardGroup>
